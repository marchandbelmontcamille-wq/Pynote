Guide

Installation en 3 étapes

Aucune connaissance technique requise.

  1. 1
    Télécharge l'installateur

    Clique sur le bouton "Télécharger" et récupère Pynote-Setup-*.exe depuis les releases GitHub.

  2. 2
    Lance l'installateur

    Double-clique et suis le wizard. Si Windows affiche une alerte SmartScreen, clique sur "Informations complémentaires" → "Exécuter quand même".

  3. 3
    Connecte-toi

    Saisis l'URL Pronote de ton établissement, ton identifiant et ton mot de passe. La session est mémorisée pour les prochains lancements.
